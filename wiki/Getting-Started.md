# Getting Started

> **Goal:** get the app running on your phone and signed in as yourself — in about 3 minutes.

---

## What is the Jetonomy App?

The Jetonomy App is a mobile app for your Jetonomy community. Your members install it on their phones and use it to post, reply, vote, and get notifications — all on your own WordPress site.

You do not sign up for any new service. The app connects directly to your site using WordPress's own login system.

---

## What you need

| Requirement | Details |
|-------------|---------|
| A WordPress site | With the **Jetonomy** plugin installed and active |
| Jetonomy 1.6.0 or later | Earlier versions work for login but won't serve branding |
| Your WordPress username and an Application Password | See [Signing In](Signing-In) — takes 30 seconds to create |
| A phone | iPhone or Android |

That's it. You do not need a developer, an App Store account, or any paid service just to try the app.

---

## The 3-minute path

### Step 1 — Install Expo Go on your phone

Expo Go is a free app that lets you run the Jetonomy App without publishing it to the App Store or Google Play. Think of it as a preview screen.

- **iPhone:** search "Expo Go" in the App Store, or visit [expo.dev/go](https://expo.dev/go)
- **Android:** search "Expo Go" in the Google Play Store

### Step 2 — Create an Application Password on your WordPress site

An Application Password is a safe, separate password WordPress generates for the app. It is not your real WordPress password.

1. Log in to your WordPress dashboard.
2. Go to **Users → Profile** (or **Users → All Users → your name**).
3. Scroll down to the **Application Passwords** section.
4. Type a name in the box (for example: `Jetonomy App`).
5. Click **Add New Application Password**.
6. WordPress shows a password — copy it now. You will not see it again.

Keep this code safe. You'll paste it into the app in Step 3.

> See the full walkthrough with screenshots on the [Signing In](Signing-In) page.

### Step 3 — Run the app and sign in

Follow the [Try the App](Try-the-App) page to launch the app from the GitHub repository using Expo Go. When the sign-in screen appears:

- **Site address:** your site's URL (for example `https://mycommunity.com`)
- **Username:** your WordPress username
- **Password:** the Application Password you just copied

You're in. Admins and moderators will see a **Manage** tab for moderation tools.
