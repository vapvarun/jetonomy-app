# Features

The Jetonomy App exposes the features that are active on your WordPress site. Enable an extension in Jetonomy → Pro Extensions and it appears in the app automatically — no app update needed.

---

## For members — Free (Jetonomy core)

| Feature | Details |
|---------|---------|
| **Activity feed** | Browse posts sorted by Hot, New, or Top across all spaces |
| **Spaces** | Forums, Q&A boards, and Idea boards — each with its own rules and sort order |
| **Post and reply** | Write posts, reply to threads, format with Markdown |
| **Voting** | Upvote and downvote posts and replies |
| **Accept answers** | Post authors can mark the best reply as the accepted answer (Q&A spaces) |
| **Search** | Full-text search across posts, replies, and spaces |
| **Notifications** | In-app alerts for replies, mentions, and votes |
| **Bookmarks** | Save posts to read later |
| **Profile** | View and edit your own profile; view other members' profiles |
| **Multiple communities** | Sign in to more than one Jetonomy community from the same app |

---

## For members — Pro (requires Jetonomy Pro)

These features appear in the app only when the matching extension is enabled in your WordPress dashboard under **Jetonomy Pro → Extensions**.

| Feature | Extension to enable |
|---------|-------------------|
| **Private messaging** | Private Messaging |
| **Emoji reactions** | Reactions |
| **Polls** | Polls |
| **Badges** | Badges |
| **Custom member fields** | Custom Fields |
| **Native push notifications** | Push Notifications (also requires a real app build — not Expo Go) |

---

## For admins and moderators

Admins and users with a moderator role see a **Manage** tab in the app. This area is not visible to regular members.

| Tool | What it does |
|------|-------------|
| **Moderation queue** | Review posts and replies flagged for moderation before they go public |
| **Flags** | See reports submitted by members; approve, dismiss, or act on them |
| **Announcements** | Pin announcements visible to all members across the community |
| **Analytics** | View high-level community activity stats |

---

## How features appear automatically

When you activate or deactivate a Jetonomy Pro extension in WordPress, the app detects the change the next time it fetches community data. You do not need to release a new app build or ask members to update anything.
